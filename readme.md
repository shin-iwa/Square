## "BookSquare"とは？
#### アプリケーション"BookSquare"は、ブックレビュー投稿サイトです。

アカウント登録の後、全ての機能をご利用頂けます。
- ブックレビュー閲覧  
- ブックレビュー投稿（及び修正・削除）
